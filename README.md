# ProjectClover (プロジェクト・クローバー)

Unityを使用した物語体験型ゲームプロジェクトです。

## プロジェクト概要
このプロジェクトは、30日間のループの中で真実を探求するシナリオを中心としたゲーム開発を目指しています。

## ディレクトリ構造
- **Assets/**: ゲームのアセット（シーン、スクリプト、設定など）が格納されています。
  - **Assets/Scenes/**: ゲームの各場面（シーン）ファイル。
  - **Assets/Settings/**: レンダリング（URP）や入力システムの設定。
- **Docs/**: (今後作成) シナリオ設定や設計仕様書などのドキュメントを日本語で管理します。
- **Library/Packages/**: Unity MCP (uloopmcp) などのパッケージが含まれています。

1.  **日本語での解説**: ドキュメントや解説テキストは日本語で作成し、分かりやすさを重視します。

3.  **シナリオ重視**: ループと分岐を含む、深い物語体験を提供します。
